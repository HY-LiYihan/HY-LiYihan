安装与配置

从零开始搭建开发环境，安装依赖并运行系统。

环境要求

项目要求推荐
操作系统Ubuntu 22.04 / 24.04Ubuntu 22.04 LTS
ROS 版本ROS2 Humble / JazzyHumble Hawksbill
Python3.10+3.10
CUDA11.8+（GPU 部署时）12.2
内存16 GB32 GB
GPURTX 3060+（推理时）RTX 4070
⚠️ 注意
如果你使用的是 Jetson 平台，请确保已安装 JetPack 6.0+，并使用对应的 CUDA 版本。

克隆仓库

# 克隆主仓库
git clone https://github.com/your-org/your-project.git
cd your-project

# 初始化子模块
git submodule update --init --recursive

Docker 安装（推荐）

使用 Docker 是最简单的安装方式，可以避免环境冲突问题。

构建镜像

# 构建 ROS2 开发环境镜像
docker compose build dev

# 构建部署镜像（含 TensorRT）
docker compose build deploy

启动容器

# 启动开发环境
docker compose up dev

# 在另一个终端进入容器
docker compose exec dev bash
💡 提示
如果你想在容器中使用 GUI 工具（如 RViz2、rqt），请确保主机已配置 X11 转发。详见 开发环境 章节。

手动安装

如果不使用 Docker，也可以手动安装依赖。

安装 ROS2

# 设置 ROS2 源
sudo apt update && sudo apt install curl -y
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key \
  -o /usr/share/keyrings/ros-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) \
  signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] \
  http://packages.ros.org/ros2/ubuntu \
  $(. /etc/os-release && echo $UBUNTU_CODENAME) main" \
  | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

# 安装 ROS2 Humble
sudo apt update
sudo apt install ros-humble-desktop -y

安装项目依赖

# 安装 Python 依赖
pip install -r requirements.txt

# 安装 ROS2 工作空间依赖
cd ros_ws
rosdep install --from-paths src --ignore-src -r -y

验证安装

# Source 环境
source /opt/ros/humble/setup.bash
source install/setup.bash

# 运行自检
ros2 run your_package self_check

如果看到以下输出，说明安装成功：

[INFO] [self_check]: All modules loaded successfully ✓
[INFO] [self_check]: Perception module  — OK
[INFO] [self_check]: Navigation module  — OK
[INFO] [self_check]: Control module     — OK
[INFO] [self_check]: System ready.
❌ 常见问题
如果 rosdep 安装失败，请检查网络连接，或尝试设置代理：
export ROSDISTRO_INDEX_URL=https://mirrors.tuna.tsinghua.edu.cn/rosdistro/index-v4.yaml
